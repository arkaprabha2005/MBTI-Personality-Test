# MBTI Personality Test App 

A Flutter-based personality test application inspired by the MBTI (Myers–Briggs Type Indicator) questionnaire.  
The app presents a 70-question assessment and evaluates the user’s personality type based on their responses.

## Features

- 70-question MBTI-style test
- Automatic personality evaluation (E/I, S/N, T/F, J/P)
- Clean UI with gradient backgrounds and card-based layout
- Start screen, quiz screen, and result screen flow
- Responsive design for different screen sizes
- Custom app icon and native splash screen

## Screens

- Start screen for beginning the test
- Quiz screen with one question at a time
- Result screen showing the final personality type

## Tech Stack

- Flutter
- Dart
- Material UI
